How to use

Typing. Pick an input system, or just start typing — the site auto-detects Devanagari, IAST, Baraha (~M ~g ~j Ru Dx), and iTrans (~N ~n RRi .D). Picking a system by hand turns detection off; clearing the box turns it back on. You can also drag & drop a .brh, .itx, or .txt file onto the input box.

Alternate spellings. Write maMtra (mantra) to send maMtra to the Indic output (मंत्र) and the parenthesized mantra to the roman output. Multi-word parens replace the same number of words before them.

Meanings. Start a line with > to attach an English meaning to the verse above it. Meanings are never transliterated; they appear in every view and become the meaning paragraph in Word/print exports. Verses are separated by blank lines.

Sharing. Share link copies a URL that opens with the current text and settings; your last session is restored automatically. Red-marked letters in the Indic output are characters that did not transliterate — usually a typo.

Saving & exports. Save keeps named texts in your browser. Word (.docx) and Print / PDF export side-by-side tables (Devanagari | roman) per verse, with meanings spanning below. Syllabify inserts syllable hyphens directly into the input (kar-pU-ra-gau-raM): the Devanagari output ignores them, the roman output and exports keep them, and you can edit them right in the input box. Unhyphenated lines still get an automatic split at export.

Vedic accents (iTrans or Baraha input) — type the mark right after the vowel:

TypeMarkExample
\'udatta अ॑agni\'m → अग्नि॑म्
\_anudatta अ॒I\_Le → ई॒ळे
"double svarita tA"t → तात्

Accents combine with anusvara and visarga: ta\_M → तं॒, gna\'H → ग्नः॑. The IAST accents and the double svarita use glyphs specific to the Sanskrit 2003 font — the font notice below the outputs will tell readers to install it.
